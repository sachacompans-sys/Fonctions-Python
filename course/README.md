Collection d'Exercices et Exemples Python
Bienvenue dans ce dépôt ! Il s'agit d'une collection de scripts Python variés, conçus pour pratiquer et illustrer différents concepts fondamentaux de la programmation.

Objectif
Ce dépôt sert de bac à sable pour explorer divers aspects du langage Python, de la syntaxe de base à des concepts plus avancés comme la programmation orientée objet.

Thèmes Abordés
Les scripts de ce projet couvrent les sujets suivants :

Algorithmes de base : Implémentation de logique comme le "FizzBuzz" (fizzbuzz_mod.py), le chiffrement de César (caesar_cipher.py) et des jeux simples comme le "Plus ou Moins" (more_or_less.py).

Programmation Orientée Objet (POO) : Utilisation des classes et des objets pour modéliser des concepts du monde réel, comme un compte bancaire (bank_account.py) ou une voiture (car.py).

Manipulation de Données :

Strings : Comptage de la fréquence des lettres (count_letters.py).

Listes : Filtrage de doublons (filter_double.py).

Dictionnaires : Stockage de données structurées (customers.py).

Scripts Utilitaires : Création de petits outils pratiques, comme un convertisseur d'unités (converter.py).

Interactions Utilisateur : Utilisation de input() pour créer des scripts interactifs.

Gestion de Fichiers : Lecture de données depuis un fichier texte (main.py).
